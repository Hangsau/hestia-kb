# 短池 100m 蛙泳：轉身佔 44%——訓練優先順序大顛覆

## 研究問題
短池 100m 蛙泳各區段（start、turn、clean swimming、finish）對總時間的貢獻與關鍵變數。

## 方法
- 15名男性泳者，自動比賽分析系統
- 分段：start（0-15m）、turn（wall前5m至出水後10m）、finish（95-100m）、clean swimming
- 統計：雙變量相關與偏相關

## 關鍵發現

**各區段時間佔比：**

| 區段 | 佔總時間比 |
|---|---|
| Turns | **44.30%** |
| Clean swimming | 38.93% |
| Start | 11.39% |
| Finish | 5.36% |

**與總時間的相關性：**
- Start 段時間：r=0.979（幾乎完全相關）
- Clean swimming 時間：r=0.940
- 10m turn-out 時間：r=0.829

**劃水頻率（cycle rate）與划距（cycle length）與 clean swimming 時間不相關。**

## 教練意涵

**蛙泳訓練優先順序：Turns > Clean swimming > Start > Finish**

這個數據將許多教練的直覺完全翻轉：

1. **轉身是最大時間消耗（44%）**：不是划水，是轉身。訓練時間應該大幅分配在轉身
2. **起跳與總時間的相關性極高（r=0.979）**：起跳爆發力幾乎決定成敗
3. **10m pull-out 水下動作品質是獨立關鍵**：不是姿勢，是過渡速度
4. **划水頻率和幅度不是 clean swimming 的瓶頸**：表示蛙泳的問題不在水面上

**實務方向：**
- 陸上訓練：爆發力（起跳、推蹬）
- 水中訓練：水下 pull-out 的速度維持 + 轉身反應
- 划水技術：不是不重要，而是當轉身和起跳已經優化到極致時，才是最後的提升空間

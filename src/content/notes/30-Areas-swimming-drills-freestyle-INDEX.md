# Freestyle Drills

> 基於 Cokie Lepinski, *There's A Drill For That!* (2011)
> 原始 PDF：Fr01 – Fr24，共 24 個 drill

---

## 常見缺陷 → 對應 Drill

| 常見缺陷 | 對應 Drill |
|----------|-----------|
| 只在一邊呼吸 / 兩邊呼吸不平均 | [[Fr01]] |
| 呼吸時頭部過度旋轉（看向天空） | [[Fr01]], [[Fr11]], [[Fr23]] |
| 頭部不在線（太高/太低） | [[Fr02]], [[Fr04]], [[Fr13]] |
| 上下身脫節（snakes/zigzags） | [[Fr04]], [[Fr06]], [[Fr09]], [[Fr23]] |
| 不會用核心旋轉 | [[Fr04]], [[Fr06]], [[Fr09]], [[Fr11]], [[Fr23]], [[Fr24]] |
| 划手不對稱（影響平衡） | [[Fr10]], [[Fr11]], [[Fr15]], [[Fr17]], [[Fr18]], [[Fr19]] |
| 踢腿不良 / 呼吸時腳外擺 | 所有踢腿 drill：[[Fr02]]–[[Fr09]], [[Fr11]], [[Fr18]], [[Fr23]] |
| 划手越過身體中心線 | [[Fr13]], [[Fr15]], [[Fr19]] |
| 划手退後過遠（出手在大腿而非屁股） | [[Fr12]], [[Fr19]] |
| 划手進入過早（划手過短） | [[Fr06]], [[Fr15]], [[Fr17]] |
| 抓水時手不下沉 anchor | [[Fr11]], [[Fr14]], [[Fr15]], [[Fr24]] |
| 進入時手腕或掌心先入水（drop elbow） | [[Fr12]], [[Fr17]], [[Fr18]], [[Fr19]] |
| 手和手臂在水下越過身體 | [[Fr13]], [[Fr14]], [[Fr16]], [[Fr19]] |
| 划水時手掌拍打水面 | [[Fr21]], [[Fr22]] |
| 每划距離不良（過多或過少划數） | [[Fr20]] |

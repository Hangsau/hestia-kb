# Grab Start vs Track Start：最佳 set-up 角度 40°/40°–50°/50°

## 研究問題
Grab Start（GS）與 Track Start（TS）兩種起跳技術的 block phase 生物力學，以及不同性別的最佳 set-up 位置。

## 方法
- 26男 / 24女 精英泳者
- 兩種技術（GS、TS）最大努力起跳
- 8個 block phase 變數：後向峰值力、峰值力、平均力、達峰時間、block time、前向脈衝、下向脈衝、起跳速度
- 5個表現變數：5m/10m 時間、水平距離、飛行時間、起跳角
- 統計：雙因子 ANOVA + 逐步多元迴歸

## 關鍵發現

**性別差異（效應量大，ηp² ≥ 0.40）：**
- 男性在兩種技術中：block phase 力量更大、block time 更快、起跳速度更高、水平距離更遠
- 只有男性：block phase 變數與表現變數顯著相關

**最佳 set-up 角度（男女適用於兩種技術）：**
- Lower leg angle（下腿角度）：40°–50°
- Whole-body angle（全身角度）：40°–50°
- Grab Start 與 Track Start 適用相同角度範圍

## 教練意涵

**個體化 set-up 幾何是起跳優化的核心。**

不論 GS 或 TS，教練可以用這個角度範圍做初始參考，再根據泳者身形調整：

- [ ] 測量泳者起跳時 lower leg 角度（池邊蹲姿時小腿與地面）
- [ ] 測量全身前傾角度
- [ ] 目標：兩個角度都在 40°–50° 區間

**性別差異的訓練啟示：**
- 男性：針對性 block phase 力量訓練對表現有直接幫助
- 女性：block phase 力量與表現的連結不如男性顯著 → 訓練重心應放在水下技術與過渡階段

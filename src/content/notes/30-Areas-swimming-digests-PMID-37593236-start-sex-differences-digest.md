# 起跳性別差異：男性 block time 快 10%，且 block 變數只對男性有效

## 研究問題
男女泳者起跳表現與生物力學的差異，以及 block phase 變數對兩性的預測力是否相同。

## 方法
- 26男（18.0±2.6歲）/ 24女（17.3±2.0歲）精英泳者
- 儀器化起跳平台，3D 運動學數據
- 變數：5m/10m 時間、水平距離、飛行時間、起跳角、block time、峰值力、達峰時間、平均力、起跳速度

## 關鍵發現

**Block time：**
- 男性：0.61 ± 0.07s
- 女性：0.67 ± 0.07s（**慢 10%**，p<0.001）

**Time to peak force（力量達峰時間）：**
- 女性：0.26 ± 0.04s
- 男性：0.23 ± 0.05s（**女性慢 10%**）

**重要發現：Block phase 變數與表現的相關性是性別專屬的。**
- 男性：block phase 變數與起跳表現顯著相關
- 女性：block phase 變數與表現**無顯著相關**（需從其他面向切入）

## 教練意涵

| 對象 | 訓練重點 |
|---|---|
| 男性泳者 | Block phase 力量訓練直接有效（峰值力、block time 都是可訓練的瓶頸） |
| 女性泳者 | Block phase 變數預測力有限 → 訓練重心應放在水下動作、過渡速度等其他面向 |

**女性起跳訓練的新方向：**
- Time to peak force 較慢 → Plyometrics（跳躍訓練）可能有幫助
- Block time 落後 → 反應爆發力訓練（不只練力量，練「碰到牆就爆發」的反應速度）

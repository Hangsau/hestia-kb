# PMID-41780505 — Key Contributors to Top-Elite Swimming Performance: Differences Between Junior and Adult Female Freestyle Swimmers.

**Journal:** Int J Sports Physiol Perform | **Date:** 2026 Jun 1  
**PMID:** 41780505 | **PMC:**   
**Relevance:** 10/10

## Abstract

**PURPOSE**: 調查 start、turn、clean-swimming 表現對總成績的貢獻，在頂尖青少年與成人女子游泳員中。

**METHODS**: 681 場長池比賽影片分析（歐洲青少年與成人錦標賽，50-1500m 含預賽/準決賽/決賽）。

**RESULTS**:
- Start、turn、clean-swimming 表現隨距離非線性發展
- Start times 從 50m 漸增至 400m，800m 開始平台
- Turn times 從 50-200m 增加，400m 開始平台
- Clean-swimming speed 從 50-200m 下降，400m 開始平台
- **Clean-swimming speed 對總時間貢獻最大**
- Start 表現對短距離（sprint）貢獻最大
- Turn 表現呈倒 U 型，200-400m 貢獻最高
- 成人 vs 青少年：50m start 顯著快 6.03%；200m start 快 5.86%
- 100m turn 年齡差距 4.92%；200m turn 差距 6.49%（隨距離遞減）
- Clean-swimming 差距：50m 3.17%、100m 1.71%、長距離 1.23-1.34%
- 百分位參考值已整合成可預測與調控總時間的軟體工具

**CONCLUSIONS**: 本研究幫助教練識別個別選手的相對強/弱項。**Clean-swimming 速度是最大可控因素**；start 與 turn 對短距離至關重要，但對長距離效用遞減。

## Notes

- **與 2026-06-15 已消化 PMID-35755613（歐洲泳者表現發展）高度互補**：本篇量化 start/turn/clean 三段貢獻度，PMID-35755613 描述整體趨勢
- **教練應用**：對 50/100m 短距離選手優先投資 start 訓練；200m 選手應同時強化 turn；400m+ 長距離 clean-swim 效率是主戰場

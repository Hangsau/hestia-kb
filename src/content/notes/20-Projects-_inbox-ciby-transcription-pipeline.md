# Ciby 語音轉寫管線

> Ciby 專案的語音轉寫子專案。

## 目標

建立可重現的語音轉寫管線，支援多格式音檔的自動轉換、段落分割與批量處理。

## 技術方案

- 使用 ffmpeg 進行音訊格式轉換
- Python 3.12.10 + uv 環境管理
- 批量處理佇列

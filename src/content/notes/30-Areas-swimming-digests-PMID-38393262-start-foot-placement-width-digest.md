# 起跳腳位：窄站優於中站和寬站

## 研究問題
技能泳者的起跳腳位寬度（窄/中/寬）對表現的影響。

## 方法
- 19名技能泳者
- 三種腳位條件：窄、中、寬
- 測量：block time、2m 時間、5m 時間、總時間

## 關鍵發現

**窄腳位**在 block time、2m 時間、5m 時間均顯著優於中站和寬站（p<0.05）。

中站和寬站之間**沒有顯著差異**。

## 教練意涵

起跳教學的第一步：確認腳位夠窄。

窄站為什麼有效？
- 窄站讓臀部與腿部的爆發力傳遞更直接
- 寬站產生的橫向分力分散了水平推進力

實務檢查清單：
- [ ] 泳者起跳前腳位是否盡量與髖同寬（或略窄）
- [ ] 腳尖朝正前方，不外開
- [ ] 前後腳重心分配均勻

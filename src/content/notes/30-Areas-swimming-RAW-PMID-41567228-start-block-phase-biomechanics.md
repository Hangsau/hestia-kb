## Abstract (RAW — English only)

### BACKGROUND
The purpose of this study was to determine the relationship between block phase biomechanics and starting performance in grab start (GS) and track start (TS) from different starting positions (sex differences).

### METHODS
Twenty-six male and 24 female elite swimmers performed maximal effort starts in GS and TS. Eight-block phase variables were measured: rearward peak force, peak force, average force, time to peak force, block time, forward impulse, downward impulse, and take-off velocity. Five performance variables were measured: 5 m swim time, 10 m swim time, horizontal distance, flight time, and take-off angle. Two-way ANOVA was used to assess differences between sexes and start techniques, and stepwise multiple regression was used to identify the best predictors.

### RESULTS
(1) Male swimmers had significantly higher block phase forces, faster block times, higher take-off velocities, and longer horizontal distances than female swimmers in both start techniques. Effect sizes were large (ηp² ≥ 0.40). (2) Males showed significantly stronger correlations between block phase forces and performance variables compared with females. (3) Optimal set-up positions were 40°/40°–50°/50° for lower leg/whole-body angles in both start techniques.

### CONCLUSIONS
These results highlight the importance of sex-specific strength training and individualized optimal set-up positions when developing start technique.

## Key Findings
1. **Sex differences in block phase**: Males showed significantly larger block phase forces, faster block times, higher take-off velocities, longer horizontal distances vs females (ηp² ≥ 0.40) — large effect sizes
2. **Sex differences in correlations**: Only male swimmers showed significant correlations between block phase variables and performance variables
3. **Optimal angles**: 40°/40°–50°/50° for lower leg/whole-body angles in both grab start and track start
4. **Performance variables**: 5m swim time, 10m swim time, horizontal distance, flight time, take-off angle
5. **Block phase variables**: rearward peak force, peak force, average force, time to peak force, block time, forward impulse, downward impulse, take-off velocity

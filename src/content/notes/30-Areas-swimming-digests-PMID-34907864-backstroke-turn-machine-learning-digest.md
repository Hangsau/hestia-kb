# 機器學習預測仰泳轉身：四種技術各有關鍵變數

## 研究問題
用機器學習預測青少年仰泳轉身表現，並找出每種技術的最重要輸入變數。

## 方法
- 120名年齡組泳者（12-18歲）
- 四種轉身技術：open、somersault、bucket、crossover
- 64個運動學/時間/動力學變數
- 模型：Lasso、Ridge、AdaBoost

## 關鍵發現

**AdaBoost 預測力最佳：R² = 0.644–0.825**

**四種技術各有不同預測變數：**

| 技術 | 最重要變數 |
|---|---|
| Open | 髖外展肌峰值扭矩時間 |
| Somersault | 手入水時的水平速度 |
| Bucket | 踝關節活動度 |
| Crossover | 膝伸肌峰值扭矩時間 |

## 教練意涵

**沒有哪種技術更好，只有哪種更適合這個人。**

每種 turn 技術的表現驅動因子不同，訓練必須個體化：

- **Open turn → 強化髖外展肌爆發力**
- **Somersault → 優化手入水時的水平速度**
- **Bucket → 提升踝關節活動度（拉筋/鬆動術）**
- **Crossover → 強化膝伸肌爆發力**

R² 0.644–0.825 的預測力代表：教練可根據泳者偏好的轉身技術，針對其關鍵變數設計訓練，而不是用同一套內容套所有技術。

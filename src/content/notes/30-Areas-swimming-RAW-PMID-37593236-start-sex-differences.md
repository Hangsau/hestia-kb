## Abstract (RAW — English only)

### OBJECTIVE
To examine sex differences in start performance and kinematics in competitive swimmers using instrumented starting platforms.

### METHODS
Twenty-six male (age: 18.0 ± 2.6 years; height: 1.83 ± 0.06 m; mass: 76.4 ± 6.5 kg) and 24 female (age: 17.3 ± 2.0 years; height: 1.70 ± 0.05 m; mass: 63.3 ± 5.6 kg) elite swimmers performed maximal effort starts. Three-dimensional kinematic data were collected using an instrumented starting platform. Performance variables (5 m swim time, 10 m swim time, horizontal distance, flight time, and take-off angle) and kinematic variables (block time, backward and forward peak forces, time to peak force, average force, and take-off velocity) were compared between sexes using one-way MANOVA.

### RESULTS
Male swimmers showed significantly higher values (p < 0.05, ηp² ≥ 0.40) for peak forces, average force, take-off velocity, horizontal distance, flight time, and lower 5 and 10 m swim times. Block time was 10% lower in males (0.61 ± 0.07 vs. 0.67 ± 0.07 seconds; p < 0.001). Time to peak force was 10% longer in females (0.26 ± 0.04 vs. 0.23 ± 0.05 seconds). Only in the male group were significant correlations (p < 0.05) found between block phase variables and start performance.

### CONCLUSIONS
The results indicate that male swimmers have superior start performance and suggest that relationships between block phase variables and start performance are sex-specific. Training should account for these sex-related differences.

## Key Findings
1. **Sex differences in block phase**: Males had significantly higher peak forces, average force, take-off velocity, horizontal distance, flight time; lower 5m and 10m swim times
2. **Block time**: Males 10% faster (0.61 ± 0.07s vs 0.67 ± 0.07s, p < 0.001)
3. **Time to peak force**: Females 10% longer (0.26 ± 0.04s vs 0.23 ± 0.05s) — slower force development
4. **Correlations sex-specific**: Only males showed significant correlations between block phase variables and start performance
5. **Effect sizes**: Large (ηp² ≥ 0.40) for all sex differences

# 起跳關鍵因子：Block phase 64% + Flight phase 65%

## 研究問題
精英衝刺泳者起跳表現的關鍵決定因子是什麼？Block phase 和 flight phase 誰更重要？

## 方法
- 14名男性精英衝刺泳者，最大努力起跳
- 3D 動作捕捉 + 測力台數據
- 變數：block phase（前後力、峰值力時間、block time、起跳角）與 flight phase（飛行時間、水平距離、水平速度）
- 統計：相關性 + 多元迴歸

## 關鍵發現

| 變數 | 對 5m 時間的解釋方差 |
|---|---|
| Block phase | **64%** |
| Flight phase | **65%** |

**最重要變數：**
- **後腿膝角度（back knee angle）** 在池邊起身時
- **Block time**（來自峰值力大小與達峰時間）
- **起跳角（take-off angle）**

## 教練意涵

**兩者都關鍵，訓練必須雙軌並進。**

Block phase 訓練重點：
- 後腿蹲距深度與膝角度（起動爆發力的幾何前提）
- 峰值力的產生速度（time to peak force）

Flight phase 訓練重點：
- 飛行軌跡執行（不是姿勢，是幾何）
- 起跳角度優化

**顛覆迷思**：過去常強調「入水角度」，但研究顯示 block phase 和 flight phase 各佔 64% / 65% 变异量，兩者是不可分割的連續體，入水只是一連串優化的結果。
